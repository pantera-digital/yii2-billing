# Yii2 billing module

## Install

Run

```
composer require pantera-digital/yii2-billing
```

or add to composer.json

```
"pantera-digital/yii2-billing": "*",
```

and execute:

```
php composer update
```

After that run migrations:

```
php yii migrate --migrationPath=vendor/pantera-digital/yii2-billing/migrations
```
